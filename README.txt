Running the Game
----------------

You need to Python and pygame installed.

Open a terminal / console and "cd" to the game directory and run:

	cd lunarpatrol
	python lunarpatrol

Under Windows, if Python installed, double-click on lunarpatrol\lunarpatrol.py
should be sufficient.

 
How to Play the Game
--------------------

Cursor keys and space bar needed. Steer spaceship with cursor keys and fire
with spacebar.


Development notes 
-----------------

Creating a source distribution with::

   python setup.py sdist

You may also generate Windows executables and OS X applications::

   python setup.py build
